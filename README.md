# Password Generator

Piccola applicazione web che genera password casuali con JavaScript e mostra il risultato direttamente nella pagina.

## Funzionalità
- generazione di una password casuale
- password di 12 caratteri
- inclusione di lettere maiuscole, minuscole, numeri e simboli
- interfaccia semplice con un pulsante per creare una nuova password

## Tecnologie usate
- HTML
- CSS
- JavaScript

## Come usare il progetto
1. Clona o scarica il repository
2. Apri `index2.html` nel browser
3. Clicca su **Generate password**
4. Visualizza la password generata a schermo

## Struttura del progetto
- `index2.html` → struttura della pagina
- `index2.css` → stile dell’interfaccia
- `index2.js` → logica di generazione della password

## Note
La password viene generata in modo casuale combinando caratteri maiuscoli, minuscoli, numeri e simboli.
